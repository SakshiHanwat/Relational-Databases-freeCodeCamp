# Some Good VS Code Extensions You Can Use in Your Editor

## Introduction

Visual Studio Code becomes much more powerful with the help of **extensions**. Extensions add extra features to the editor and help improve productivity, readability, and overall coding experience. This document explains some popular and useful VS Code extensions in easy English for notes and revision.

---

## Better Comments

**Better Comments** helps make your comments more meaningful and visible.

Features:

* Highlights special comments like **TODO**
* Uses different colors for questions, warnings, and important notes
* Makes comments easier to notice while reading code

---

## Code Spell Checker

**Code Spell Checker** checks spelling mistakes in your code.

Features:

* Detects spelling errors in variable names, comments, and strings
* Understands coding styles like **camelCase**
* Designed specially for code files, not normal documents

---

## Error Lens

**Error Lens** makes errors more visible in your code.

Features:

* Highlights the **entire line** where an error occurs
* Shows the error message directly on the line
* Easier than relying only on underlines provided by VS Code

---

## Indent Rainbow

**Indent Rainbow** adds colors to different levels of indentation.

Features:

* Each indentation level has a different color
* Helps understand code structure and scope
* Works very well with VS Code’s bracket pair colorization

---

## AI Assistants (GitHub Copilot / Tabnine)

AI assistants help by suggesting code while you type.

Examples:

* **GitHub Copilot**
* **Tabnine**

Features:

* Inline code suggestions
* Helps write code faster
* Useful for learning patterns and syntax

---

## Icon Packs

Icon packs improve the look of the file explorer.

Example:

* **VS Code Great Icons**

Features:

* Adds meaningful icons to files and folders
* Makes the file tree cleaner and easier to understand

---

## Colorize

**Colorize** helps understand colors used in CSS.

Features:

* Shows actual color previews for CSS color values
* Makes it easier to work with design-related code

---

## Language-Specific Extensions

Some extensions are specific to certain languages.

### JavaScript Extensions

* **ESLint** – Checks code quality and finds bugs
* **Prettier** – Automatically formats code

### TypeScript Extensions

* **Pretty TypeScript Errors** – Shows clearer and easier-to-read error messages

---

## Fun Extensions

VS Code also has extensions just for fun and motivation.

### VS Code Pets

* Adds virtual pets inside the editor
* Keeps you company while coding

### Power Mode

* Creates flashy effects when you write code continuously
* Makes coding feel more energetic

### Discord Presence

* Shows your current coding activity on Discord
* Lets friends see what you are working on

---

## Conclusion

VS Code extensions allow you to customize your editor based on your needs. From productivity tools to fun extensions, there are many options available. You should explore the extension marketplace and choose what works best for your workflow.

---

## Summary

* Extensions add extra features to VS Code
* Some improve readability and error detection
* Some help with specific languages
* Some are just for fun
* Choose extensions based on your project needs
