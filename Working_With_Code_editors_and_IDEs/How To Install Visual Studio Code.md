# How to Install Visual Studio Code on Your Computer

## Introduction

Visual Studio Code (VS Code) is a popular, lightweight code editor used by developers all over the world. It supports many programming languages and works on Windows, macOS, and Linux. This guide explains **how to install VS Code step by step** on different operating systems in easy English. These notes are suitable for beginners and for revision.

---

## Important Note Before Installing

When downloading VS Code, it is very important to:

* **Visit the Visual Studio Code website**, not the Visual Studio website.

Visual Studio and Visual Studio Code are different products. Visual Studio Code is a code editor, while Visual Studio is a full IDE.

---

## Installing VS Code on Windows

1. Go to the official **VS Code website**.
2. Download the **EXE installer** for Windows.
3. Once the file is downloaded, double-click on it.
4. The **setup wizard** will open.
5. Follow the on-screen instructions to complete the installation.
6. After installation, you can open VS Code from the Start Menu.

---

## Installing VS Code on macOS

There are multiple ways to install VS Code on a Mac.

### Method 1: Using Zip Archive

1. Download the **.zip archive** from the VS Code website.
2. Extract the zip file.
3. You will get a **.app file**.
4. Double-click the .app file to run VS Code.

### Method 2: Using Homebrew

If you use **Homebrew** on macOS, you can install VS Code using the Terminal.

1. Open the **Terminal** application.
2. Run the following command:

```
brew install --cask visual-studio-code
```

3. After the installation is complete, you can open VS Code from Applications.

---

## Installing VS Code on Linux

1. Visit the VS Code website.
2. Download either a **.deb** or **.rpm** file depending on your Linux distribution.
3. Install the downloaded package using your systemâ€™s package manager.
4. Once installed, you can launch VS Code:

   * From the applications menu, or
   * Using the **command line** (usually by typing `code`).

---

## Launching Visual Studio Code

After installation on any operating system:

* You can open VS Code using the desktop or applications menu.
* You can also launch it from the **command line** using the `code` command (if enabled).

Now your system is ready for coding.

---

## Summary

* Always download VS Code from the **VS Code website**, not Visual Studio.
* Windows uses an **EXE installer**.
* macOS supports **zip archive** and **Homebrew** installation.
* Linux supports **.deb** and **.rpm** packages.
* VS Code can be launched using GUI or command line.

These steps help you install VS Code correctly on any major operating system.
